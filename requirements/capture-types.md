# Capture Input Types

The following content types are supported:

1. Text
2. Audio
3. Image
4. Video

## Allowed Formats

Implementations may support any subset of formats appropriate to their capture source.

### Text

- Unicode UTF-8

### Audio

- m4a
- flac
- oga

### Image

- jpeg
- png

### Video

- mp4
- mkv
- webm
